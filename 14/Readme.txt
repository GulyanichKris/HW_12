Тебе необходимо написать свой первый класс, который будет описывать структуру пользователя (можешь взять любую другую структуру из жизни — еду, товар, животное), у которого будет следующий набор свойств — имя, фамилия, возраст, пол.

Создай как минимум два объекта от этого класса с разными значениями, и выведи на экран (можно через var_dump, можно через echo отдельные поля)

Результат задания можешь приложить файлом, или ссылку на репозиторий с данными заданием.
